## Service Level Objectives

Strait tracks reliability signals from workflow and run execution. These signals are used for dashboards, alerts, and SLO reporting.

| Metric | What it measures |
|---|---|
| `success_rate` | Percentage of runs that complete without error |
| `p95_latency_secs` | 95th percentile run duration in seconds |
| `p99_latency_secs` | 99th percentile run duration in seconds |

SLO windows are evaluated over configured intervals such as **24 hours**, **7 days** (168h), or **30 days** (720h), depending on the deployment and analytics configuration.

## API Surface

| Endpoint | Purpose |
|---|---|
| `GET /v1/analytics/performance` | Review run duration, throughput, and success trends. |
| `GET /v1/usage/anomalies` | List detected usage and spend anomalies. |
| `GET /v1/anomaly-config` | Read anomaly detection thresholds. |
| `PUT /v1/anomaly-config` | Update anomaly detection thresholds. |
| `GET /metrics` | Export Prometheus metrics for external alerting. |

## Anomaly Detection

Strait monitors spending patterns across your usage and flags unusual spikes. Two severity levels:

- **Warning** — spending exceeded the warning threshold compared to the recent baseline
- **Critical** — spending exceeded the critical threshold

## Configuring anomaly detection

Set your thresholds with a multiplier. A `warning_threshold` of `1.5` means spending 1.5x above the baseline triggers a warning.

```bash
curl -X PUT https://api.strait.dev/v1/anomaly-config \
  -H "Authorization: Bearer $STRAIT_API_KEY" \
  -d '{
    "warning_threshold": 1.5,
    "critical_threshold": 3.0
  }'
```

Retrieve current configuration:

```bash
curl https://api.strait.dev/v1/anomaly-config \
  -H "Authorization: Bearer $STRAIT_API_KEY"
```

## Checking for anomalies

```bash
curl https://api.strait.dev/v1/usage/anomalies \
  -H "Authorization: Bearer $STRAIT_API_KEY"
```

```json
{
  "anomalies": [
    {
      "detected_at": "2025-01-15T08:30:00Z",
      "severity": "warning",
      "current_spend": 42.50,
      "baseline_spend": 25.00,
      "ratio": 1.7
    }
  ]
}
```

An empty `anomalies` array means spending is within normal bounds.
