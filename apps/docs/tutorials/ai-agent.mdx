## What You'll Build

An AI document summarization agent that:

- Processes documents through GPT-4o and returns summaries
- Reports token usage and cost for every run
- Enforces a **$5 per-run limit** so no single document blows the budget
- Enforces a **$100 daily budget** across all runs in the project
- Emits a warning at 80% of daily spend so you can react before hitting the hard cap

## Prerequisites

- Strait running with a project created (`proj_1`)
- TypeScript SDK installed: `npm install @strait/ts`
- An OpenAI API key with access to GPT-4o

## Step 1: Set Cost Budgets

Configure spending limits on your project. Strait tracks costs in **micro-USD** (1 USD = 1,000,000 micro-USD) for precision without floating-point drift.

```bash
curl -X PUT https://api.strait.dev/v1/projects/proj_1/quota \
  -H "Authorization: Bearer $API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "max_cost_per_run_microusd": 5000000,
    "max_daily_cost_microusd": 100000000,
    "timezone": "America/New_York"
  }'
```

This sets:

- **$5 per run** (5,000,000 micro-USD). Any single run that reports more than this is failed with a budget error.
- **$100 per day** (100,000,000 micro-USD). Once the project hits this, new triggers are rejected until the next calendar day in the configured timezone.
- At **80% ($80)**, Strait emits a warning event and metric so you can adjust before the hard cutoff.

## Step 2: Define the Agent Job

```typescript
import { createClient, defineJob } from "@strait/ts";
import OpenAI from "openai";

const client = createClient({
  baseUrl: "https://api.strait.dev",
  auth: { type: "bearer", token: process.env.STRAIT_API_KEY! },
});

const openai = new OpenAI({ apiKey: process.env.OPENAI_API_KEY! });

interface SummarizePayload {
  document_id: string;
}

interface SummarizeResult {
  summary: string;
  token_count: number;
  cost_microusd: number;
}

const summarizer = defineJob({
  name: "Document Summarizer",
  slug: "summarize-doc",
  endpointUrl: "https://my-api.com/jobs/summarize",
  projectId: "proj_1",
  maxAttempts: 2,
  timeoutSecs: 600,
  run: async (payload: SummarizePayload, ctx): Promise<SummarizeResult> => {
    ctx.logger.info("Summarizing document", { docId: payload.document_id });

    const doc = await fetchDocument(payload.document_id);

    const response = await openai.chat.completions.create({
      model: "gpt-4o",
      messages: [
        {
          role: "system",
          content: "Produce a concise summary of the following document.",
        },
        { role: "user", content: doc.text },
      ],
    });

    const usage = response.usage!;
    const costMicrousd = calculateCost(usage);

    await ctx.reportUsage({
      provider: "openai",
      model: "gpt-4o",
      prompt_tokens: usage.prompt_tokens,
      completion_tokens: usage.completion_tokens,
      cost_microusd: costMicrousd,
    });

    await ctx.reportProgress(100, "Summary complete");

    return {
      summary: response.choices[0].message.content!,
      token_count: usage.prompt_tokens + usage.completion_tokens,
      cost_microusd: costMicrousd,
    };
  },
});
```

The `reportUsage` call is what connects your LLM spend to Strait's budget enforcement. Without it, Strait has no visibility into what a run actually cost.

## Step 3: Calculate Costs

Map OpenAI's token counts to micro-USD using the published pricing for GPT-4o.

```typescript
function calculateCost(usage: {
  prompt_tokens: number;
  completion_tokens: number;
}): number {
  // GPT-4o pricing: $2.50 per 1M input tokens, $10.00 per 1M output tokens
  const inputCost = (usage.prompt_tokens / 1_000_000) * 2_500_000;
  const outputCost = (usage.completion_tokens / 1_000_000) * 10_000_000;
  return Math.round(inputCost + outputCost);
}
```

Example: a 4,000-token input and 500-token output costs `(4000/1M * 2.5M) + (500/1M * 10M)` = 10,000 + 5,000 = **15,000 micro-USD ($0.015)**.

## Step 4: Register, Trigger, and Check Cost

```typescript
async function main() {
  // Register the job definition with Strait
  await summarizer.register(client, { projectId: "proj_1" });

  // Trigger a run
  const run = await summarizer.trigger(client, {
    payload: { document_id: "doc_123" },
  });
  console.log("Run started:", run.id);

  // Poll for completion (in production, use webhooks instead)
  let result;
  do {
    await new Promise((r) => setTimeout(r, 2000));
    result = await client.getRun({ pathParams: { runID: run.id } });
  } while (result.status === "queued" || result.status === "running");

  console.log("Status:", result.status);
  console.log("Cost:", result.usage?.cost_microusd, "micro-USD");

  if (result.status === "completed") {
    console.log("Summary:", result.result?.summary);
  } else {
    console.error("Failed:", result.error);
  }
}

main().catch(console.error);
```

```typescript
// Helper used by the job — replace with your actual document store
async function fetchDocument(id: string): Promise<{ text: string }> {
  const res = await fetch(`https://my-api.com/documents/${id}`);
  if (!res.ok) throw new Error(`Document ${id} not found`);
  return res.json();
}
```

## What Happens at the Limit

**Per-run limit ($5):** If a single run's cumulative `reportUsage` calls exceed 5,000,000 micro-USD, Strait fails the run immediately with a `budget_exceeded` error. The partial result is discarded. This prevents a runaway document (e.g., a 200-page PDF) from burning your entire daily budget in one shot.

**Daily limit ($100):** Once total reported cost across all runs in the project reaches 100,000,000 micro-USD for the calendar day, Strait rejects new `trigger` calls with a `429` status. Runs already in progress are allowed to finish. The counter resets at midnight in the configured timezone.

**80% warning ($80):** When daily spend crosses 80,000,000 micro-USD, Strait emits a warning event and increments a metric. Wire this to your alerting system so you can pause non-critical work or raise the limit before the hard cutoff hits.
