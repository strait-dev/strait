The Strait MCP server (`@strait/mcp-server`) gives any MCP-compatible AI agent native access to the full Strait API. Instead of writing SDK code, your agent discovers and calls 133 tools, reads 5 live resources, and uses 18 prompt templates, all through natural language.

Works with Claude Desktop, Claude Code, Cursor, Windsurf, VS Code Copilot, OpenAI Codex CLI, and any other MCP-compatible client.

## Get Started

Run the server with your API key:

```bash
STRAIT_API_KEY=strait_live_... npx @strait/mcp-server
```

Then connect your preferred AI client using the setup instructions below.


## Configuration

The server reads configuration from three sources, in this order of precedence: CLI flags, environment variables, then `strait.json`.

### Config file

Create a `strait.json` in your project root (optional):

```json
{
  "apiKey": "strait_live_...",
  "baseUrl": "https://api.strait.dev",
  "transport": "stdio",
  "port": 3000
}
```

See [Configuration](/sdks/configuration) for the full `strait.json` schema.

### Options reference

| Option | Env Var | CLI Flag | Config Key | Default |
|--------|---------|----------|------------|---------|
| API Key | `STRAIT_API_KEY` | `--api-key` | `apiKey` | (required) |
| Base URL | `STRAIT_BASE_URL` | `--base-url` | `baseUrl` | `https://api.strait.dev` |
| Transport | | `--transport` | `transport` | `stdio` |
| Port | | `--port` | `port` | `3000` |
| Config file path | | `--config` | | `./strait.json` |

<Note>
  The API key is optional when OAuth is configured via `OAUTH_ISSUER_URL` and `OAUTH_RESOURCE_URL`.
</Note>

## Architecture

```text
MCP Client <-- stdio or HTTP --> @strait/mcp-server (Node.js) -- REST --> Strait API
```

Built with [Effect TS](https://effect.website), Zod 4, and OpenTelemetry. Every tool call is traced, entity IDs are annotated on spans, errors are typed, and retries use exponential backoff.

To export traces, set `OTEL_EXPORTER_OTLP_ENDPOINT` to your collector endpoint.

## Troubleshooting

| Error | Fix |
|-------|-----|
| `STRAIT_API_KEY is required` | Set via env var, `--api-key` flag, or `strait.json` |
| `NetworkError: API unreachable` | Check `STRAIT_BASE_URL` (default: `https://api.strait.dev`) |
| `ApiError: 401 Unauthorized` | API key expired or missing scopes. Regenerate at [strait.dev](https://strait.dev). |
| `ApiError: 404 Not Found` | Resource does not exist. Use the list tool to find available resources. |
| `RateLimitError: 429` | Automatic retry with backoff. Reduce request frequency if the error persists. |
| MCP client not detecting server | Check config path, check that `npx` is on your PATH, restart your IDE |
