The Rust SDK repository is [strait-dev/strait-rust](https://github.com/strait-dev/strait-rust).

Do not add a `strait` crate from crates.io for Strait Cloud. As of the last verification for these docs, crates.io does not publish a Strait Cloud crate named `strait`.

## Current Status

| Item | Status |
|---|---|
| Source repository | Available |
| crates.io package | Not published as `strait` |
| Recommended production path | Use the REST API or another verified SDK |

## Use the REST API From Rust

Until a Strait-owned crate is published and verified, call the REST API directly.

```rust
use reqwest::Client;
use serde_json::json;

#[tokio::main]
async fn main() -> Result<(), Box<dyn std::error::Error>> {
    let base_url = std::env::var("STRAIT_BASE_URL")
        .unwrap_or_else(|_| "https://api.strait.dev".to_string());
    let api_key = std::env::var("STRAIT_API_KEY")?;

    let response = Client::new()
        .post(format!("{base_url}/v1/jobs/send-welcome-email/trigger"))
        .bearer_auth(api_key)
        .json(&json!({
            "payload": {
                "user_id": "user_abc123",
                "email": "customer@example.com"
            }
        }))
        .send()
        .await?
        .error_for_status()?;

    let run: serde_json::Value = response.json().await?;
    println!("{} {}", run["id"], run["status"]);

    Ok(())
}
```
