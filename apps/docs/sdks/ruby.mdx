The Ruby SDK repository is [strait-dev/strait-ruby](https://github.com/strait-dev/strait-ruby).

Do not install the `strait` gem from RubyGems for Strait Cloud. As of the last verification for these docs, that RubyGems package is unrelated to Strait Cloud.

## Current Status

| Item | Status |
|---|---|
| Source repository | Available |
| RubyGems package | Not verified for Strait Cloud |
| Recommended production path | Use the REST API or another verified SDK |

## Use the REST API From Ruby

Until a Strait-owned RubyGems package is published and verified, call the REST API directly.

```ruby
require "json"
require "net/http"
require "uri"

base_url = ENV.fetch("STRAIT_BASE_URL", "https://api.strait.dev")
api_key = ENV.fetch("STRAIT_API_KEY")

uri = URI("#{base_url}/v1/jobs/send-welcome-email/trigger")
request = Net::HTTP::Post.new(uri)
request["Authorization"] = "Bearer #{api_key}"
request["Content-Type"] = "application/json"
request.body = JSON.dump({
  payload: {
    user_id: "user_abc123",
    email: "customer@example.com"
  }
})

response = Net::HTTP.start(uri.hostname, uri.port, use_ssl: uri.scheme == "https") do |http|
  http.request(request)
end

raise response.body unless response.code.start_with?("2")

run = JSON.parse(response.body)
puts "#{run["id"]} #{run["status"]}"
```
