The Strait CLI is the Go CLI shipped from `apps/strait/cmd/strait`.

The canonical implementation lives in:
- `apps/strait/cmd/strait/`
- `apps/strait/internal/cli/`

## Current direction

> Status: Go CLI is the shipped command surface.

- One CLI surface: `strait`
- Go implementation in `apps/strait`
- Script-safe operational commands for jobs, runs, workflows, events, deploys, and RBAC
- Deterministic manifest builds from `strait.json` or `strait.config.yaml`/`.yml`
- Deployment versions with create, finalize, promote, rollback, and list flows

## Interaction policy

> Default behavior is deterministic and automation-safe.

- Non-interactive commands must remain machine-consumable.
- `--json` output must emit a single JSON document when advertised as script-safe.
- Interactive prompts are opt-in and must be disabled in CI or non-TTY contexts.

## Current CLI baseline

The Go command set in `apps/strait/cmd/strait` is the source of truth for shipped behavior.

Key command groups:
- `jobs`, `runs`, `workflows`, `workflow-runs`, `events`
- `build`, `deploy`, `project`, `secrets`
- `perf`, `team`, `audit`, `status`, `doctor`

## Installation

```bash
# Build from source (from repo root)
go build -o strait ./apps/strait/cmd/strait

# Or install from the app directory
cd apps/strait && go install ./cmd/strait
```
