Use the Runs API to operate work after it has been triggered.

For request body fields generated from the runtime OpenAPI schema, see [Generated API Fields](/api-reference/generated-fields#runs).

## Prerequisites

```bash
export STRAIT_BASE_URL=https://api.strait.dev
export STRAIT_API_KEY=strait_replace_with_your_key
export STRAIT_RUN_ID=run_replace_with_your_run_id
```

## Inspect a Run

```bash
curl -sS "$STRAIT_BASE_URL/v1/runs/$STRAIT_RUN_ID" \
  -H "Authorization: Bearer $STRAIT_API_KEY" | jq
```

Expected result: a run object with `status`, `attempt`, and timestamps.

## Read Run Events

```bash
curl -sS "$STRAIT_BASE_URL/v1/runs/$STRAIT_RUN_ID/events" \
  -H "Authorization: Bearer $STRAIT_API_KEY" | jq
```

## Replay a Run

```bash
curl -sS -X POST "$STRAIT_BASE_URL/v1/runs/$STRAIT_RUN_ID/replay" \
  -H "Authorization: Bearer $STRAIT_API_KEY" | jq
```

## Common Operations

| Operation | Method and path |
|---|---|
| List runs | `GET /v1/runs` |
| List dead-letter runs | `GET /v1/runs/dlq` |
| Bulk cancel all active runs | `POST /v1/runs/bulk-cancel-all` |
| Get run | `GET /v1/runs/{runID}` |
| Cancel run | `DELETE /v1/runs/{runID}` |
| Replay run | `POST /v1/runs/{runID}/replay` |
| Replay dead-letter run | `POST /v1/runs/{runID}/dlq-replay` |
| List child runs | `GET /v1/runs/{runID}/children` |
| List events | `GET /v1/runs/{runID}/events` |
| List outputs | `GET /v1/runs/{runID}/outputs` |
| List checkpoints | `GET /v1/runs/{runID}/checkpoints` |
| Set debug mode | `POST /v1/runs/{runID}/debug` |
| List run lineage | `GET /v1/runs/{runID}/lineage` |
| Reset idempotency key | `DELETE /v1/runs/{runID}/idempotency-key` |
| Reschedule run | `POST /v1/runs/{runID}/reschedule` |
| Pause run | `POST /v1/runs/{runID}/pause` |
| Resume run | `POST /v1/runs/{runID}/resume` |
| Restart run | `POST /v1/runs/{runID}/restart` |
| List run state | `GET /v1/runs/{runID}/state` |
| List run resources | `GET /v1/runs/{runID}/resources` |
| Stream run updates | `GET /v1/runs/{runID}/stream` |
| Get run stream chunks | `GET /v1/runs/{runID}/stream/chunks` |

## Failure Cases

| Status | Meaning |
|---|---|
| `400` | Query or replay request is invalid. |
| `401` | API key missing or invalid. |
| `404` | Run does not exist in this project. |
| `409` | Run cannot be replayed or canceled from its current state. |
