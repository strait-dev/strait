---
title: "Secrets"
description: "Manage project secrets stored in the system keychain."
---

## secrets create

Create a new secret. If no source flag is provided, you will be prompted interactively.

| Flag | Type | Default | Description |
| :--- | :--- | :--- | :--- |
| --from-env | string | | Name of the environment variable to read the secret from |
| --from-file | string | | Path to a file to read the secret from |
| --project | string | | Project ID |

<Note>
Secrets are stored in the system keychain and are scoped per project.
</Note>

```bash
strait secrets create MY_SECRET --project my-project --from-env LOCAL_VAR
```

## secrets list

List secrets for a project.

| Flag | Type | Default | Description |
| :--- | :--- | :--- | :--- |
| --project | string | | Project ID |

```bash
strait secrets list --project my-project
```

## secrets delete

Delete a secret by name.

| Flag | Type | Default | Description |
| :--- | :--- | :--- | :--- |
| --project | string | | Project ID |

```bash
strait secrets delete MY_SECRET --project my-project
```

<Warning>
Secret injection into jobs requires the environment variable `FF_SECRET_INJECTION=true` to be set.
</Warning>
